# Репрайсер WB 3.0 — Daily Notes

## 2026-05-10 — Создание проекта

### Инициализация
- **Название:** Репрайсер WB 3.0
- **Технологии:** Google Sheets + Apps Script + Wildberries API
- **Статус:** Планирование

### Требования от Фёдора

#### Таблица данных (столбцы 1-16):
1. Первое фото товара
2. Артикул ВБ
3. Артикул продавца
4. Название товара
5. Бренд
6. РРЦ (цена в клиентском приложении)
7. Модель удержания (С кошельком ВБ / Без кошелька ВБ)
8. Цена до скидки продавца ВБ
9. Скидка продавца (дробью)
10. Цена со скидкой продавца
11. Цена с клиентского сайта без кошелька (парсинг)
12. Скидка ВБ (расчёт: 11 / 9)
13. Цена с кошельком (парсинг)
14. % кошелька (расчёт: 14 / 11)
15. Загружаемая цена в ЛК
16. Статус загрузки

#### API и токены:
- **API ключ:** хранить в настройках Apps Script
- **Токены API WB:**
  - API токен 1: только чтение (загрузка контента и товаров)
  - API токен 2: чтение + запись (передача цен)
- **Токен авторизованного пользователя:** для парсинга цен с клиентского сайта
- **Лист настроек:** отдельный лист для токенов и настроек

#### Документация:
- Отдельный лист с инструкцией

#### Тайминг запуска репрайсера:
- Выбор частоты: раз в 30 мин / раз в час / раз в 2 часа
- Настройка триггеров в Apps Script
