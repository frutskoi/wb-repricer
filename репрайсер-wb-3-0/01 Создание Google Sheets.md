# Этап 1: Создание Google Sheets для Репрайсер WB 3.0

## 📋 Шаг 1: Создание новой таблицы

1. Открой [sheets.google.com](https://sheets.google.com)
2. Нажми "Пустой файл" или "+" для создания новой таблицы
3. Назови таблицу: `Репрайсер WB 3.0`

## 📋 Шаг 2: Создание 4 листов

### Лист 1 — Товары (переименуй из "Лист1")
1. Кликни на вкладку "Лист1" → "Переименовать"
2. Название: `Товары`
3. Скопируй заголовки из файла `template_products.csv` в строку 1

### Лист 2 — Настройки
1. Нажми "+" внизу для создания нового листа
2. Название: `Настройки`
3. Скопируй содержимое из `template_settings.csv`

### Лист 3 — Лог
1. Создай новый лист
2. Название: `Лог`
3. Скопируй содержимое из `template_log.csv`

### Лист 4 — Инструкция
1. Создай новый лист
2. Название: `Инструкция`
3. Скопируй содержимое из этого файла

## 📋 Шаг 3: Настройка формата столбцов (Лист "Товары")

### Столбец A (Фото) — формат текста
- Выдели столбец A → Формат → Число → Обычный текст

### Столбец B (Артикул WB) — число
- Выдели столбец B → Формат → Число → Число (без десятичных)

### Столбец C (Артикул продавца) — текст
- Выдели столбец C → Формат → Число → Обычный текст

### Столбец F (РРЦ) — число
- Выдели столбец F → Формат → Число → Валюта → ₽

### Столбцы H, J, K, M, O (цены) — валюта
- Выдели столбцы H, J, K, M, O → Формат → Число → Валюта → ₽

### Столбец I (Скидка продавца) — процент
- Выдели столбец I → Формат → Число → Процент

### Столбец L (Скидка WB) — процент
- Выдели столбец L → Формат → Число → Процент

### Столбец N (% кошелька) — процент
- Выдели столбец N → Формат → Число → Процент

### Столбец G (Модель удержания) — выпадающий список
1. Выдели столбец G (начиная с G2)
2. Данные → Проверка данных
3. Правила: "Список из диапазона"
4. Введи: `С кошельком ВБ,Без кошелька ВБ`
5. Нажми "Готово"

### Столбец P (Статус загрузки) — выпадающий список
1. Выдели столбец P (начиная с P2)
2. Данные → Проверка данных
3. Введи: `В очереди,Загружено,Ошибка,В процессе`
4. Нажми "Готово"

## 📋 Шаг 4: Настройка формул (Лист "Товары")

### Столбец L (Скидка WB)
В ячейку L2 вставь формулу:
```
=IFERROR(ROUND((K2/I2-1)*100, 2), 0)
```
Протяни формулу вниз для всего столбца.

### Столбец N (% кошелька)
В ячейку N2 вставь формулу:
```
=IFERROR(ROUND(M2/K2*100, 2), 0)
```
Протяни формулу вниз для всего столбца.

## 📋 Шаг 5: Условное форматирование (Лист "Товары")

### Статус загрузки (столбец P)
1. Выдели столбец P (начиная с P2)
2. Формат → Условное форматирование
3. Правила:
   - Если "Загружено" → зелёный фон (#d4edda)
   - Если "Ошибка" → красный фон (#f8d7da)
   - Если "В очереди" → жёлтый фон (#fff3cd)
   - Если "В процессе" → синий фон (#d1ecf1)

### Разница цен (когда будет столбец 17)
После добавления столбца 17 настроим условное форматирование для визуализации разницы между РРЦ и фактической ценой.

## 📋 Шаг 6: Защита настроек (Лист "Настройки")

1. Выдели весь лист "Настройки"
2. Данные → Защитить лист и диапазоны
3. Выбери "Лист: Настройки"
4. Нажми "Установить разрешения"
5. Ограничь редактирование (опционально)

## 📋 Шаг 7: Поделиться таблицей

1. Нажми "Поделиться" в правом верхнем углу
2. Добавь тех, кто будет работать с репрайсером
3. Установи права: "Редактор"

## ✅ Проверка после выполнения

После завершения всех шагов проверь:

- [ ] Все 4 листа созданы и переименованы
- [ ] Заголовки в листе "Товары" соответствуют плану (16 столбцов)
- [ ] Форматы столбцов настроены (текст, число, валюта, процент)
- [ ] Выпадающие списки работают в столбцах G и P
- [ ] Формулы в столбцах L и N работают
- [ ] Условное форматирование для статусов работает
- [ ] Лист "Настройки" содержит все параметры
